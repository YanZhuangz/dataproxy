DataProxy
==================

DataProxy 是一个基于 [Arrow Flight](https://arrow.apache.org/docs/format/Flight.html) 的数据服务框架，接入丰富的数据源，提供统一、易用、高效、健壮的数据读写服务。通过 DataProxy：

* 你可以接入丰富的数据源，其中包括 MySQL、S3、Aliyun OSS、本地磁盘等
* 你可以使用统一的接口来实现对不同数据源的读写操作
